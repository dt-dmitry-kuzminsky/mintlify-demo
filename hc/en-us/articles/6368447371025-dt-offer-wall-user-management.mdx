## User Management

The User page displays all team members and their specific roles.

<Frame caption="mceclip5.png">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/8063901900689/mceclip5.png" />
</Frame>[](/hc/article_attachments/6368443504273/Screen_Shot_2020-06-03_at_9.10.33.png)

The User Management Users window also details for those users with a developer role, the publishers and apps that they have permission to access.

Use the search function to either search for a specific team member or for a particular role by entering text in the search field.

## Inviting a New User

Follow these steps to add a new user:

1. Click **User Management** on the left side toolbar\
   <Frame caption="user_management.png">
     <img src="https://developer.digitalturbine.com/hc/article_attachments/10554989656989" />
   </Frame>

2. Click <Frame caption="mceclip0.png">
     <img src="https://developer.digitalturbine.com/hc/article_attachments/9340805933725" />
   </Frame>

3. The **Invite New User** window opens.\
   Enter the name and email of the new User

4. Select the role for the new users

5. Click**Send Invite****<Frame caption="Screen_Recording_2023-02-15_at_14.39.25.mp4-high.gif">
     <img src="https://developer.digitalturbine.com/hc/article_attachments/9341616928669" />
   </Frame>**

The new user receives an email with a link to activate their account.

The new user is now able to perform tasks in the DT dashboard in accordance with the role permissions granted by the admin.

This article is for publishers who want to use the DT FairBid SDK to load and display ads from Verve Group via DT FairBid mediation.

Verve Group currently supports the following [ad types](https://developer.digitalturbine.com/hc/en-us/articles/360010915358-Ad-Types-and-Placement-ID):

* Banner
* Interstitial
* Rewarded

To integrate Verve Group as a mediated network with DT :

* [Step 1: Retrieve Verve Group parameters](#h_01J53A0485SWFD9YTTCPS0RKCE).
* [Step 2: Add Verve Group as a DT mediated network](#h_01J53A0485TXZ5BR1APYGVR5NW).
* [Step 3: Add the Verve Group SDK to your DT integration](#h_01J53A04855WHGBFQVH4M7ENX7).
* [Step 4: Testing your integration](#h_01JD6A13TQ41ABKN1HTXP3EJTR).

## Step 1: Retrieve Verve Group Parameters

Before retrieving Verve Group parameters, ensure that you have Verve Group ad unit inventory that you want to mediate with DT. For more information about creating Verve Group inventory, see the following Verve Group articles:

* [Create](https://dashboard.pubnative.net/) a Verve Group account.
* [Add](https://developers.verve.com/reference/hybid-cloud-documentation#step-2a-add-application) your app to Verve Group.
* [Create](https://developers.verve.com/reference/hybid-cloud-documentation#step-2b--add-placement) zones (placements) for your app in Verve Group.
* If you want to mediate bidding placements, [add bidders](https://developers.verve.com/reference/hybid-cloud-documentation#step-3-add-bidders) to your placements.

To add Verve Group as a DT mediated network, locate the following parameters from your Verve Group account:

| VERVEGROUP PARAMETER | REQUIREDFOR              | DESCRIPTION                                                                                                                                       |
| --------------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Reporting API Key`   | Waterfall                 | Unique access token for your app in the Verve Group system for reporting purposes. In the DT Console, use this value as the `Account Auth Token`. |
| `App ID`              | Waterfall                 | Unique identifier for your app in the Verve Group system.                                                                                         |
| `App Auth Token`      | * Waterfall
* SDK Bidding | Unique access token for your app in the Verve Group system. In the DT Console, use this value as the `App Token`.                                 |
| `Zone Number`         | - Waterfall
- SDK Bidding | Unique identifier for the placement in the Verve Group system.                                                                                    |

## Obtaining the Reporting API Key

The Verve Group `Reporting API Key` serves as the Account Auth Token for DT.

To obtain the Verve `Reporting API Key`:

1. Log in to your [Verve Group dashboard](https://dashboard.pubnative.net/), and click the drop-down next to your account name.

2) Click **Reporting API Key**.\
   The Reporting API Key is copied.

<Frame caption="Verve Group Reporting API Key">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/5432203086993" />
</Frame>

3. Store this value in a text file. This value is required later in [Step2](#h_01J53A0485TXZ5BR1APYGVR5NW "Step 2: Adding Verve Group as a DT mediated network") to add Verve Group as a DT mediated network.

## Obtaining the App ID

To obtain the Verve Group `App ID`:

1. Log in to your [Verve Group dashboard](https://dashboard.pubnative.net/).\
   The dashboard URL changes to contain your Verve Group `App ID`.

<Frame caption="Verve Group App ID">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/5432233098385" />
</Frame>

2. Store this value in a text file. This value is required later in [Step2](#h_01J53A0485TXZ5BR1APYGVR5NW "Step 2: Adding Verve Group as a DT mediated network") to add Verve Group as a DT mediated network.

## Obtaining the App Auth Token

1. From your [Verve Group dashboard](https://dashboard.pubnative.net/), click **Apps**, and then click the pencil icon for your app.\
   The **Edit App** page displays the `App auth token`.

<Frame caption="Verve Group App auth token">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/5432417005073" />
</Frame>

2. Copy the `App auth token`, and store this value in a text file. This value is required later in [Step2](#h_01J53A0485TXZ5BR1APYGVR5NW "Step 2: Adding Verve Group as a DT mediated network") to add Verve Group as a DT mediated network.

## Obtaining the Zone Number

To obtain the Verve `Zone Number`:

1. From your [Verve Group dashboard](https://dashboard.pubnative.net/), click **Apps**, and then click the gear icon for your app.\
   A list of zones associated with your app displays.

<Frame caption="Verve Group Zone Number">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/6490308753425" />
</Frame>

2. For each Zone (placement) you want to mediate with DT, copy the `Zone Number`, and store these values in a text file. These values are required later in [Step2](#h_01J53A0485TXZ5BR1APYGVR5NW "Step 2: Adding Verve as a DT mediated network") to add Verve Group as a DT mediated network.

## Step 2: Add Verve Group as a DT Mediated Network

Before you add Verve Group as a mediated network in DT FairBid:

* Ensure that you have access to parameters you retrieved from Verve Group in [Step1](#h_01J53A0485SWFD9YTTCPS0RKCE).
* [Add your App](https://developer.digitalturbine.com/hc/en-us/articles/14755391313308-Adding-an-App) using the DT Console.
* Ensure that you set up a [DT Placement](https://developer.digitalturbine.com/hc/en-us/articles/15005035293468-Setting-Up-Placements "Create a DT placement") for each Verve Group ad type you want to mediate with DT.

To add Verve Group as a DT mediated network:

1. Log in to the [DT Console](https://console.fyber.com/), and click**App Management**.

2) [Set up](https://developer.digitalturbine.com/hc/en-us/articles/15005016082972-Setting-Up-Mediated-Networks) **Verve** or **Verve Bidder** as a mediated network for your app using the parameters you retrieved in [Step1](#h_01J53A0485SWFD9YTTCPS0RKCE).

3. [Create an ad network instance](https://developer.digitalturbine.com/hc/en-us/articles/15111147606428-Setting-Up-Instances) for each placement you want to mediate with DT. Use the Verve Group `Zone Number` you retrieved in [Step1](#h_01J53A0485SWFD9YTTCPS0RKCE) as the **Instance ID**.

## Step 3: Add the Verve Group SDK to your Integration

To integrate the Verve Group SDK, follow the instructions on the [Supported Networks page](https://developer.digitalturbine.com/hc/en-us/articles/360010077777-Supported-Networks).

## Step 4: Test Your Integration

Use the DT Test Suite to verify that you have set up your app properly for FairBid mediation. The DT Test Suite is available for Android, iOS, and Unity apps. For more information about using the DT Test Suite, see [Test Suite](https://developer.digitalturbine.com/hc/en-us/articles/16840582428316-Test-Suite "Using the DT Test Suite").

## Legend

| **Field**            | **Description**                                                                                                                                                                                                                                                                                              |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Dashboard - Account  | The value is configured via network's dashboard, configurable on account level                                                                                                                                                                                                                               |
| "IsChild" API        | The "IsChild" API is designed to help publishers comply with age-related policies like COPPA, GDPR, Google Play’s Families Ads Program, and Apple App Store Kids policy. For more information, click [here](https://developer.digitalturbine.com/hc/en-us/articles/6995888598813-Flag-End-Users-as-a-Child). |
| Dashboard - App      | The value is configured via network's dashboard, configurable on app level                                                                                                                                                                                                                                   |
| DT Placement         | The value is configured via DTdashboard, configurable on placement level                                                                                                                                                                                                                                    |
| Client Side          | The value is set only on the client side in the source code of the app                                                                                                                                                                                                                                       |
| Client Side (Shared) | The value is set only on the client side via shared preferences (IAB standard)                                                                                                                                                                                                                               |
| Passed via FairBid   | The value is set via FairBid SDK API                                                                                                                                                                                                                                                                         |
| N/A                  | Not Supported / Unknown                                                                                                                                                                                                                                                                                      |

## Getting Started

ACP Edge is a self-serve campaign management platform that allows advertisers to manage user acquisition campaigns within DT's Offer Wall. It enables advertisers to monitor results in real time, adjust bids based on performance by country and source, and manage campaigns through a single dashboard.

To begin setting up your campaign, add a product:

1. Navigate to the [ACP Edge](https://acp-edge.fyber.com/login) login page and log in to your account.\
   The main **Dashboard** appears.

<Frame caption="">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/8095992337809" />
</Frame>

2. Click **Add Product** from the ACP Edge dashboard.
3. Select one of the following product types:**Mobile App** or**Web**.
4. For **Mobile App** products, enter the **App Name** or **Developer Name **and select the app from the **Google Play** or **App Store.**

<Frame caption="">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/360011838578" />
</Frame>

4. Click **Save**.
5. For **Web **products, enter the **Product Name** and upload an icon.

<Frame caption="">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/360012072537" />
</Frame>

#### Note

The app icon updates automatically from the App Store and is refreshed every 24 hours.

6. Click **Save**.
7. (Optional) To disable or remove products from the list, click <Frame caption="2024-12-23_14-55-40.jpg">
     <img src="https://developer.digitalturbine.com/hc/article_attachments/17697679300252" />
   </Frame> and select **Disable Product** or **Remove**.

## Creating a Campaign

Once you have added a product to the ACP Edge dashboard, you must configure the campaign to associate it with the product.

Campaign creation involves the following stages:

* [Setting up a Campaign](#h_01JFYDEY0YX3R5GHS3ND8V8CX2)
* [Tracking a Campaign](#h_01JG95T8SC9JR3Q6HH4799XRZE)
* [Targeting a Campaign](#h_01GQZ102FW71D5XBVTSTYMTKDH)

## Setting Up a Campaign

To create a campaign:

1. Select a product on the **Dashboard** and click **Add Campaign**.\
   The **Add Campaign **window appears.
2. Enter the required values in the **General Configuration** area, as described in [Add Campaign Parameters](#target-paragraph).

<Frame caption="2024-12-26_11-27-08.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697679303836" />
</Frame>

3. Enter the required values in the **Budget** area, as described in [Add Campaign Parameters](#target-paragraph).

<Frame caption="2024-12-26_11-27-27.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697654689180" />
</Frame>

4. Enter the required values in the **Campaign Customizations** area, as described in [Add Campaign Parameters](#target-paragraph).

<Frame caption="2024-12-26_11-27-51.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697679306268" />
</Frame>

5. Click **Next**.

### Campaign Types

The campaign type defines the options for a mobile app or web page to ensure compatibility on the selected platform. Click on one of the following options below for more information.

**Web Campaign Types**

| Task Type             | Description                                                                                                                                |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Lead Generation       | Users submit their email addresses or personal information to win a gift card or coupon.                                                   |
| Survey Completion     | Users participate in a market research survey, often starting with pre-screening questions.                                                |
| Purchase/Payment Req. | Users purchase, pay for a subscription, or deposit money (credit card input only and charged immediately).                                 |
| Registration          | Users create an account by signing up via Facebook or Email (no credit card is required).                                                  |
| Free Trial            | Users sign up to start a free trial, including the account creation and credit card input for later charges once the trial period expires. |
| Quiz                  | Users complete a short quiz.                                                                                                               |
| Donation              | Users make a donation.                                                                                                                     |

**App Campaign Types**

| Task Type            | Description                                                                                   |
| -------------------- | --------------------------------------------------------------------------------------------- |
| Download/App Install | Users download the app.                                                                       |
| In-App Action        | Users complete a specific event within an app, such as reaching a designated level in a game. |

### Creative Assets

Creative assets are the visual and multimedia elements used in campaigns. To add creative assets to your campaign:

1. Click the **Creative Set** icon.\
   The **Create Carousel** window appears.
2. Select assets from the right pane or drag new assets to the window.

<Frame caption="2024-12-26_12-17-32.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697654693660" />
</Frame>

The following table details the requirements for each type of asset:

| Asset | Accepted Format                                                             |
| ----- | --------------------------------------------------------------------------- |
| Video | Max duration: 31 seconds Min width: 300 Min height: 250 Max file size: 50MB |
| Image | Min width: 300 Min height: 250 Max file size: 50MB                          |

#### Note

You can upload up to three assets per campaign, with a maximum of one video.

3. Arrange the assets in your desired order of appearance.

<Frame caption="">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/6338223803677" />
</Frame>

4. Click **Save**.\
   The **Tracking** tab appears.

## Tracking a Campaign

Tracking campaigns collects data on impressions, clicks, and conversions to evaluate performance, optimize delivery, and meet objectives. DT integrates tracking providers to create unique tracking URLs and accurately attribute user actions to specific campaigns.

To track a campaign:

1. Select one of the following tracking providers from the **Tracking Provider** drop-down list.\
   For more information, see [Supported Attribution Providers](/hc/en-us/articles/360011062598).

* [Adjust](https://help.adjust.com/en/partner-setup/digital-turbine)
* [AppsFlyer](https://support.appsflyer.com/hc/en-us/articles/208209366-How-to-Configure-a-Campaign-with-Fyber)
* [Kochava](https://support.kochava.com/campaign-management/create-a-kochava-certified-postback/)
* [Mobile App Tracking by Tune](https://support.tune.com/hc/en-us/articles/1500008248741-Passing-Values-to-a-Partner-Conversion-URL)
* [Singular](https://support.singular.net/hc/en-us/articles/20797177013659-Digital-Turbine-Media-Attribution-Integration#2_Create_the_Singular_Tracking_Link)
* Server-to-Server - Customized integration

<Frame caption="2024-12-29_14-48-39.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697654694300" />
</Frame>

2. Generate a **Click URL** using the provider’s instructions.

3. Replace placeholders in the URL with relevant macro values.\
   See [Supported Macros](/hc/en-us/articles/360011062618) for available placeholders and further information.

<Frame caption="2024-12-29_14-55-44.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697679308956" />
</Frame>

#### Note

Make sure to replace the generic URL macro key with the proper key name.

4. Copy the **DT Edge App ID** to update your attribution provider's DT integration in the** ACP Edge App ID** field.\
   This ensures proper conversion tracking from both the old and new App IDs.
5. Add all the steps required to complete the campaign in the **Steps** **Description** area.\
   This is what end-users view on the **Campaign** info page.\
   See [Multi-Reward Campaigns](/hc/en-us/articles/360019015258) for further information.

<Frame caption="mceclip2.png">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/8108293484433" />
</Frame>

#### Tip

See our recommendations on campaign setup in the [Offer Description handboo](https://developer.digitalturbine.com/hc/en-us/article_attachments/17831809099676)[k](https://developer.digitalturbine.com/hc/en-us/article_attachments/17831809099676).

6. Click **Add Step **to add additional steps.\
   Include the step description and the **Action ID**.

#### Important

**Action ID** is mandatory only at the final step. The **Action ID** is not editable for live campaigns or campaigns that were live in the past.

7. (Optional) To add comments or additional information, enter text in the **Comments** area
8. (Optional) To translate the campaign steps, see [Setting Up a Translation](#h_01EVBPMQWXDWJ8RH88R1TE8018).
9. Click **Next**.\
   The **Targeting **tab appears.

## Targeting a Campaign

The Targeting window allows you to define specific criteria to deliver your campaign to the intended audience, which improves campaign relevance, optimizes performance, and maximizes return on investment. The targeting functionality is organized into the following sections:

* [Location Targeting](#h_01JG95T8SCMJSSS4DC9FW1KTMY)
* [App Targeting](#h_01JG95TF8MGCXD8068A3K9QSDT)
* [Device Targeting](#h_01JGB991JCR15N8X43SJ4WRY9E)
* [Manufacturer Targeting](#h_01JG9A3B2D31CY1212N219EGF3)

### Location Targeting

Location targeting allows you to specify countries to ensure your campaign reaches end-users in the most relevant areas. The default setting, **Worldwide**, targets all countries. To focus your campaign on a specific country:

1. Select a country from the **Search Countries** drop-down list.

<Frame caption="2024-12-29_12-09-14.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697679309980" />
</Frame>

#### Tip

To blocklist certain countries, select **All Countries** first, then deselect the countries you want to exclude.

### App Targeting

App targeting allows you to select apps where your campaign appears, ensuring it aligns with user interests and behaviors. The default setting, **All Apps**, includes all apps. To target your campaign on specific apps:

<Frame caption="2024-12-29_14-23-46.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697679312028" />
</Frame>

1. Select **Blocklist** or **Allowlist** from the **Targeting Type** drop-down list.
2. Enter the name of the apps or **App IDs** for the intended targeting.
3. (Optional) Copy and paste app IDs from an Excel or CSV file.

### Device Targeting

Device targeting defines the types of devices your campaign targets based on the operating system, OS version, or device type. The default setting includes all devices. To refine your campaign for specific devices:

<Frame caption="">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/8108396654609" />
</Frame>

1. Click **Browse**and select **iOS** or **Android**
2. To select device versions for the targeting, drag the **OS version slider**.
3. Click **Device Type** and then select **Phone** or **Tablet**.

<Frame caption="2024-12-29_15-09-36.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697654699804" />
</Frame>

4. Click **Save**.

### Manufacturer Targeting

Manufacturer targeting allows you to allow or block device brands and models to align your campaign with the target audience. The default setting includes all manufacturers. This approach optimizes bids, allowing higher bids for devices that deliver quality end-users while reducing spend on less effective ones. To allowlist or blocklist manufacturers:

1. Click **Browse** in the **Manufacturers** area and select the required **Browse Device Models**.

<Frame caption="2024-12-30_14-05-14.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697654700316" />
</Frame>

2. Select **Allowlist** or **Blocklist** in the **Targeting Type** drop-down list.
3. Click **Save**.\
   The **Preview** tab appears.

<Frame caption="Step 3- Set-Up Translations@2x.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17773994684828" />

## Setting Up Translations

In the **Campaign Customization** area, you can add translations for games localized for different countries to align campaigns with regional language preferences. Translating campaigns improves communication and engagement with multilingual audiences. DT provides the tools for adding translations; however, you are responsible for providing accurate localized content. You can translate the following elements:

* [Campaign Title](#h_01EVBPQ51QE0PFS4C2BBY8Z1P1)
* [Campaign Steps](#h_01JFYAETXE1M5W9Y0ACTR9TXZM)
* [Campaign Comments](#h_01JFYAETXECY1GVWVRQD7ZBC3D)

## Translating Campaign Title

To translate the campaign title:

1. Select a product on the **Dashboard**, then select a campaign.\
   The **Setup** window appears.
2. Click the translation icon in the **Campaign Customizations** area.\
   The **Campaign Title Translation** window appears.

<Frame caption="2024-12-30_10-27-20.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697654700828" />
</Frame>

3. Click **Add Translation**.\
   The **Campaign Title Translation** window appears.
4. Select the targetcountry for the translated campaign title from the **Select Countries** drop-down list.

<Frame caption="2024-12-30_10-14-04.jpg">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17697679315612" />
</Frame>

#### Tip

You can either select all countries and then deselect the countries you don't want, or you can individually select each country you wish for the translation.

6. Enter the translated campaign title in the **Text** field.
7. Click **Save**.

## Set Up Preview

After completing the steps to create a campaign in ACP Edge, review the setup summary and preview how the campaign will appear to end-users in a live environment.

The Preview allows you to evaluate the visual presentation of your campaign within the DT Offer Wall list and its info page on the right pane. It also includes a setup summary on the left pane to review and adjust campaign targeting.

## Opening a Preview

To preview a campaign and see how the campaign appears:

1. Click **OPEN PREVIEW**.\
   The Preview shows how the campaign appears to end-users.

#### Note

Translations do not appear in the Preview.

2. To view the options for orientation and device, click one of the following options under the image:

* Portrait
* Landscape
* Telephone
* Tablet

#### Important

The reward amount presented in this Preview is unrelated to the bid set for the campaign. Reward amounts vary between apps according to the app's exchange rate.

The following table describes the elements in the Preview.

| Element Name     | Source                                         | Customizable                                                                                                               | Field Location                                             |

The DT FairBid User Management tool allows the control of user access to areas of the dashboard and of privileges to view or perform functions within it.

## Roles and Permissions

You can choose between four different roles in the system, each with its own unique set of permissions:

* Administrator
* Manager
* Developer
* Finance

## User Management Page

1. Select **User Management** on the left hand menu\
   <Frame>
     <img src="https://developer.digitalturbine.com/hc/article_attachments/10554671436317" />
   </Frame>

2. All existing team members and their roles are displayed. The DT User Management Users window also details for those users with a developer role, the publishers and apps that they have permission to access

   <Frame caption="">
     <img src="https://developer.digitalturbine.com/hc/article_attachments/11123732328861" />
   </Frame>

3. Use the search function to either search for a specific team member or for a particular role by entering text in the search field

   <Frame caption="">

---
title: "DT Offer Wall Report For Publishers"
---

The DT Offer Wall Report is an analytic reporting tool based on the millions of events generated from DT's DT Offer Wall each day. It allows advertisers and publishers to track and improve performance by filtering and splitting data with over 40 metrics and dimensions. The Offer Wall Report for Publishers shows performance within your own app. (To view how offers perform across all apps and users in the network, see the [DT Offer Wall Report for Advertisers](https://developer.digitalturbine.com/hc/en-us/articles/360012175977-DT-Offer-Wall-Report).)

#### Tip

For more details on the DT Dynamic Reporting functionality, see [UsingtheReports](/hc/en-us/articles/17885710276124).

The DT Offer Wall Report for Publishers has two different views:

### **Performance**

Every action or revenue metric is attributed to the time of the click and impression. This view allows a clear understanding of how your manipulations to exchange rates, DT Offer Wall positions in the game, and any other action impact performance. It also allows you to see the revenue contribution from your daily active users.

### **Billing**

Every action or revenue metric is attributed to the time it happened. This view allows a clear understanding of the periodical revenues, conversions, and other events as they happened.

## DT Offer Wall Report Metrics for Publishers

The following table describes the groups and dimensions in the DT Offer Wall Report for Publishers.

| Group Name | Metric                                | Description                                                                                                                                                                              |
| ---------- | ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Delivery   | Avg. Offers per Container             | Average offers presented in DT Offer Wall container (measured by impressions). Formula: Sum(Offer Impressions) ÷ Sum(OFW Container Impressions)                                          |
|            | Avg. Position                         | Average position the Offer received for each container impression                                                                                                                        |
|            | Container Fill Rate                   | Container Impressions with at least one offer to Container Impressions rate. Formula: Container Impressions with at least one offer ÷ Container Impressions                              |
|            | Impression frequency                  | Frequency of impressions over the chosen time period.\<> \</>                                                                                                                            |
|            | Offer Impressions                     | Individual offer impression count.                                                                                                                                                       |
|            | Offer Impressions per App             | Individual offer impression count per app.                                                                                                                                               |
|            | OFW Impressions per App Active Users  | Offer impressions divided by Daily Active Users. Formula: Offer Impressions ÷ App active users                                                                                           |
|            | OFW Impressions per OFW Engaged Users | Offer impressions divided by Daily Engaged Users. Formula: Offer Impressions ÷ DEU                                                                                                       |
|            | OFW Unique Users                      | Number of unique users who opened DT Offer Wall.                                                                                                                                         |
|            | Publisher App Unique Users            | The publisher app's unique active users. This metric is accurate only when a publisher initializes our SDK on app start. When splitting by day, it shows the Daily Active Users.         |
|            | OFW Container Impressions             | DT Offer Wall container impressions count.                                                                                                                                               |
| Engagement | Container CTR                         | Offers clicks to container impressions rate. Publisher's metric. Formula: Offer click ÷ Container impression                                                                             |
|            | Conversions / Click Rate              | Offer conversions to offer clicks ratio. Formula: Offer Conversions ÷ Offer Clicks                                                                                                       |
|            | Conversions / Impressions Rate        | Offer conversions to offer impressions ratio. Formula: Offer Conversions ÷ Offer Impressions                                                                                             |
|            | Conversions / Installs Rate           | Offer conversions to the offer installs ratio. Formula: Offer Conversions ÷ Offer Installs                                                                                               |
|            | Offer Clicks                          | Refers to the clicks that take the user to the app store or landing page. When using the mobile-wall integration type, this will represent the click after the info window is displayed. |
|            | Offer Completion Time                 | Average time to complete an offer. Formula: Sum(time to convert) ÷ Sum(conversions)                                                                                                      |
|            | Offer Conversions                     | Amount of conversions generated from offers.                                                                                                                                             |
|            | Offer CTR                             | Offers clicks to impressions rate. Formula: Offer click ÷ Offer impression                                                                                                               |
|            | Offer Info Click                      | The initial clicks of the offer, which opens the offer info window. When using the DT Offer Wall API integration type, this will not be available.                                       |
|            | Offer into CTR                        | Ratio of offer cOffer to the total offer impressions. Formula: Offer info click ÷ Offer impression                                                                                       |
|            | Offer Ticket Count                    | The number of reports per offer generated by users.                                                                                                                                      |
|            | OFW Engagement Rate                   | DT Offer Wall's daily engaged users divided by the Publisher app's daily active users. Formula: OFW unique users ÷ DAU\*100                                                              |
|            | Unique Converters                     | Amount of unique users that were converted.                                                                                                                                              |
| Revenue    | Publisher ARPDAU                      | Publisher's average revenue per daily active users. This metric is accurate only when a publisher initializes our SDK on app start.                                                      |
|            | Publisher ARPDEU                      | Publisher's average revenue per daily engaged user.                                                                                                                                      |
|            | Publisher Conversions                 | Amount of conversions generated.                                                                                                                                                         |
|            | Publisher eCPA                        | Publisher's revenue measures per Publisher's payout. Formula: Sum(Publisher Revenue) ÷ Sum(Payout)                                                                                       |
|            | Publisher Net eCPM                    | Publisher's net ad revenue generated per thousand ad impressions.                                                                                                                        |
|            | Publisher Offer eCPM                  | Publisher's revenue per thousand Offer Impressions. Formula: Sum(Publisher Revenue) ÷ Sum(Offer Impressions)                                                                             |
|            | Publisher Revenue                     | Total Publisher's payout received from DT. Formula: Sum(Publisher Payout)                                                                                                                |

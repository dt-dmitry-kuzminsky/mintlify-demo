title: "Liftoff Monetize"

This article is for publishers who want to use the DT FairBid SDK to load and display ads from LiftoffMonetize via DT FairBid mediation.

LiftoffMonetize currently supports the following [ad types](https://developer.digitalturbine.com/hc/en-us/articles/360010915358-Ad-Types-and-Placement-ID):

* Banner
* Interstitial
* Rewarded

To integrate LiftoffMonetize as an DT mediated network:

* [Step 1: Locate LiftoffMonetize parameters](#h_01G3E5VEHWNAQWBHGD8RXTGQ29).

* [Step 2: Add LiftoffMonetize as a DT mediated network](#h_01G3E5WJR7101R3YZ4TQBSH2NQ):

  * [Add LiftoffMonetize for Waterfall mediation](#Waterfall).\
    This allows for traditional waterfall mediation with LiftoffMonetize. Ensure that your app and bidding placements have been added to the LiftoffMonetize network.

  * Add LiftoffMonetize for Bidding mediation:

    * [Add LiftoffMonetize as a manually managed DT mediated network](#BiddingManual).\
      This allows DT to include LiftoffMonetize in bidding mediation. Ensure that your app and bidding placements have been added to the LiftoffMonetize network.
    * [Add LiftoffMonetize as an automatically managed DT mediate network](#BiddingAuto).\
      This allows DT to include LiftoffMonetize in bidding mediation. DT adds your app and bidding placements to the LiftoffMonetize network on your behalf.

* [Step 3: Add the LiftoffMonetize/Vungle SDK to your DT integration](#h_01G3E5X6C1M6VB2SKM932Z93QK).

* [Step 4: Test your integration](#h_01JD5E71DC3JHB15ZT00X90K3Z).

## Step 1: Locate LiftoffMonetize Parameters

Depending on the type of mediation you intend to use, adding Liftoff Monetize as a mediated network requires specific parameters from the Liftoff Monetize Dashboard. Before you can locate these parameters, use the following table to ensure that you have completed any app setup tasks in the Liftoff Monetize Dashboard.

| LiftoffMonetize Tasks                                                                                                                                             | Waterfall | Bidding |           |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :-------: | :-----: | :-------: |
|                                                                                                                                                                    |           |  Manual | Automatic |
| [Create](https://support.vungle.com/hc/en-us/articles/203610124-Start-Monetizing-With-Liftoff) a LiftoffMonetize Account.                                         |          |        |          |
| [Add](https://support.vungle.com/hc/en-us/articles/115000493152-Add-and-Edit-Your-App-in-the-Dashboard) your app to LiftoffMonetize.                              |          |        |          |
| [Create](https://support.vungle.com/hc/en-us/articles/360041566292-Create-or-Edit-a-Placement) Placements in Liftoff Monetize for traditional waterfall mediation. |          |        |          |
| [Create](https://support.vungle.com/hc/en-us/articles/360041566292-Create-or-Edit-a-Placement) Liftoff Monetize placements for bidding mediation.                  |          |        |          |

To enable LiftoffMonetize as a DT mediated ad network, locate the following LiftoffMonetize parameters:

| Parameter                | Description                                                           | Waterfall | Bidding |           |
| ------------------------ | --------------------------------------------------------------------- | :-------: | :-----: | :-------: |
|                          |                                                                       |           |  Manual | Automatic |
| `App ID`                 | Unique identifier of your application in the Liftoff Monetize system. |          |        |          |
| `Placement Reference ID` | Unique identifier of the ad unit in the LiftoffMonetize system.      |          |        |          |
| `Reporting API Key`      | Your unique key for the LiftoffMonetize API.                         |          |        |          |
| `Secret Token`           | Your unique key for your LiftoffMonetize publisher self-serve API.   |          |        |          |

## Locating the App ID

From the [LiftoffMonetize Dashboard](https://publisher.vungle.com/login2), click the **Applications** drop-down and select your app.\
The** App Settings** appear and include the `App ID`.

## Step 2: Add Liftoff Monetize as a DT Mediated Network

Before you add LiftoffMonetize as a mediated network in DT FairBid, complete the following tasks in the [DTConsole](https://console.fyber.com/):

* Add your App to the DT ecosystem. For more information, see[Adding an App](https://developer.digitalturbine.com/hc/en-us/articles/14755391313308-Adding-an-App).
* Ensure that you have a DT Placement for each Mintegral ad type you want to mediate with DT. For more information, see[Setting Up Placements](https://developer.digitalturbine.com/hc/en-us/articles/15005035293468-Setting-Up-Placements "Create a DT placement").

## Adding LiftoffMonetize as Waterfall Network

To add LiftoffMonetize as a DT mediated network:

1. From the [DT Console](https://console.fyber.com/), go to **Monetization→App Management**, locate the **Network** tab for your app, and click**Addnetwork**.\
   The list of available networks appears.

<Frame caption="FB_Network-List-Liftoff_2025-04-08_11-17-18.png">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/19608201233564" />
</Frame>

2. Click the**LiftoffMonetize** tile and enter the Liftoff Monetize parameters for Waterfall integration from [Step1](#Step1).\
   For more information, see[Addinga WaterfallNetwork](/hc/en-us/articles/15005016082972-Setting-Up-Mediated-Networks#h_01JRGZPA0NGW3Q7B0PXNFGKKRR).

#### Note

Some networks offer both traditional waterfall and programmatic bidding. Networks that offer programmatic bidding appear as separate tiles labeled specifically for bidding. For waterfall mediation, do not select the *bidder* version of the network.

<Frame caption="LiftoffMonetize parameters for DT FairBid">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/10052100180765" />
</Frame>

3. Create a Waterfall network instance for each Liftoff Monetize placement you want DT to mediate. Use the Liftoff Monetize `PlacementReferenceID` you located in [Step1](#Step1) as the **InstanceID**.\
   For more information, see[Adding a Waterfall Instance](/hc/en-us/articles/15111147606428-Setting-Up-Instances#h_01JREWSZ35BVDEVSB8XDXFHKZW).

## Adding LiftoffMonetize as a Manually Managed Bidding Network

To add LiftoffMonetize as a DT mediated network:

1. From the [DT Console](https://console.fyber.com/), go to **Monetization→App Management**, locate the **Network** tab for your app, and click**Addnetwork**.\
   The list of available networks appears.

<Frame caption="FB_Network-List-Liftoff-Bidder_2025-04-08_11-17-18.png">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/19608218125084" />
</Frame>

2. Click the **LiftoffMonetizebidder** tile.\
   The **Add network** form for **LiftoffMonetizebidder** appears.\
   For more information, see[Addinga ManuallyManaged BiddingNetwork](/hc/en-us/articles/15005016082972-Setting-Up-Mediated-Networks#h_01J3NTVS4Q02JAV2B543ED1K52).

#### Note

Networks that support programmatic bidding appear as separate tiles labeled specifically for bidding. To offer your placements for bidding, make sure you select the*bidder*version of the network—in this case,**LiftoffMonetizebidder**.

<Frame caption="Liftoff-Monetize-Bidder_2025-01-14_12-12-53.png">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/17930900794140" />
</Frame>

3. Click **Manually managed **and enter the LiftoffMonetize parameters for manual management from [Step 1](#Step1).
4. Create a Bidding network instance for each LiftoffMonetize placement you want DT to mediate. Use the LiftoffMonetize `Placement Reference ID` you located in [Step1](#Step1) as the **InstanceID**.\
   For more information, see[ManuallyAdding a BiddingInstance](/hc/en-us/articles/15111147606428-Setting-Up-Instances#h_01J3RD5A45GD79RK31ERHWMV3R).

## Adding LiftoffMonetize as an Automatically Managed Bidding Network

To add LiftoffMonetize for automatically managed bidding mediation:

1. From the [DT Console](https://console.fyber.com/), go to **Monetization→App Management**, locate the **Network** tab for your app, and click**Addnetwork**.\
   The list of available networks appears.

<Frame caption="FB_Network-List-Liftoff-Bidder_2025-04-08_11-17-18.png">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/19608218125084" />
</Frame>

2. Click the **LiftoffMonetizebidder** tile.\
   The **Add network** form for **LiftoffMonetizebidder** appears.\
   For more information, see[Addingan AutomaticallyManaged BiddingNetwork](https://developer.digitalturbine.com/hc/en-us/articles/15005016082972-Setting-Up-Mediated-Networks#h_01JR8JQY9TQ9QCGT8CYV3E4HQV).

#### Note

Networks that support programmatic bidding appear as separate tiles labeled specifically for bidding. To offer your placements for bidding, make sure you select the*bidder*version of the network—in this case,**LiftoffMonetizebidder**.

<Frame caption="FB_Liftoff-Bidding-Auto_2025-04-15_12-51-24.png">
  <img src="https://developer.digitalturbine.com/hc/article_attachments/19608218126876" />
</Frame>

3. Click **Automatically managed **and enter the Liftoff Monetize parameters for automatic management from [Step 1](#Step1).
4. Auto-generate a Bidding network instance for each FairBid placement you want to mediate with LiftoffMonetize. For more information, see[AutomaticallyAdding a BiddingInstance](/hc/en-us/articles/15111147606428-Setting-Up-Instances#h_01JR13WRNMJEPRFCC26PZBZGMH).
